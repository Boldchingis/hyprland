# Hyprland

Modern desktop configuration for Hyprland with Quickshell.

## Installation
```bash
curl -fsSL https://github.com/Boldchingis/hyprland/main/install.sh | bash
```

## Features
- Application launcher with search
- Status bar with system info
- Session management
- Dynamic wallpapers
- Clean notifications

## Requirements
- Hyprland
- Qt6
- Quickshell
- Systemd
